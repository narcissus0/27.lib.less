less27
======

A LESS lib, speeding up coding.

#目录结构

```tree
.
|____css
| |____style.css            --目标文件
|____less
| |_____27
| | |_____attr.less         --属性类
| | |_____css3.less         --css3 相关
| | |_____grid.less         --删格布局
| | |_____reset.less        --css reset 相关
| | |_____lib.less          --常用样式
| | |_____oocss.less        --面向对象
| | |_____init.less         --总入口（此处调用了以上所有的库）
| |____style.less           --在此处引入进行代码编写             

```
